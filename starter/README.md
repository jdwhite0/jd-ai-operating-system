# JD AI OS — Free Starter

This is the free, stripped-down core of [JD AI OS](https://github.com/jdwhite0/jd-ai-operating-system).
Clone it, rename it, and you have your own AI operating system today.

> and domain playbooks)? Get the full product at **[jdproductions.io](https://jdproductions.io)**.

---

## Setup in 3 steps

1. **Copy this `starter/` folder** to wherever you want your system to live
   (e.g. `~/Desktop/My AI OS`).
2. **Open `AI_START_HERE.md`** and skim it — that's the boot file every AI reads.
3. **Connect your AI.** Tell it: *"Read AI_START_HERE.md first, then today.md."*
   (Claude Code reads `CLAUDE.md` automatically.)

## What's inside

| Folder | What it's for |
|--------|---------------|
| `AI_START_HERE.md` | Universal boot file — orients any AI in 2 minutes. |
| `CLAUDE.md` | Auto-loads in Claude Code. |
| `01_BRAIN/` | Who you are + what's true. Identity, priorities, domains. |
| `02_SYSTEMS/` | The rules every AI follows. |
| `03_DAILY/` | `today.md` — your current state and tasks. |
| `04_PROJECTS/` | Active work and new ventures. |
| `05_MEDIA` · `06_RESOURCES` · `07_AUTOMATIONS` · `08_BACKUPS` | Support layers. |

## Daily use

Each morning, open `03_DAILY/today.md`, set your **state** and your **top 3**. Then point
any AI at the system and go.
